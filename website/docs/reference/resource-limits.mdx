---
title: Resource Limits
---

:::info Beta feature
Resource limits are currently in beta.
:::

To ensure that Unleash operates smoothly, it includes resource limits for some of the key resources that you can create. The limit for each resource can vary between the different unleash plans, and some plans may enforce stricter limits than what the Unleash system can handle. Refer to the [Unleash pricing & plans](https://www.getunleash.io/pricing) page for more information on plan-specific limits

Those limits are:

| Resource                                               | OSS limit | Pro limit | Enterprise limit | Environment variable                           |
|--------------------------------------------------------|----------:|----------:|-----------------:|------------------------------------------------|
| [Feature flags](feature-toggles.mdx)                   |     5,000 |     5,000 |           50,000 | `UNLEASH_FEATURE_FLAGS_LIMIT`                  |
| [Projects](projects.md)                                |         1 |       500 |              500 | `UNLEASH_PROJECTS_LIMIT`                       |
| [Segments](segments.mdx)                               |       300 |       300 |              300 | `UNLEASH_SEGMENTS_LIMIT`                       |
| [API tokens](api-tokens-and-client-keys.mdx)           |     2,000 |     2,000 |            2,000 | `UNLEASH_API_TOKENS_LIMIT`                     |
| [Environments](environments.md)                        |         2 |        50 |               50 | `UNLEASH_ENVIRONMENTS_LIMIT`                   |
| [Strategies](activation-strategies.md) per environment |        30 |        30 |               30 | `UNLEASH_FEATURE_ENVIRONMENT_STRATEGIES_LIMIT` |
| [Constraints](strategy-constraints.md) per strategy    |        30 |        30 |               30 | `UNLEASH_CONSTRAINTS_LIMIT`                    |
| Values per constraint                                  |       250 |       250 |            1,000 | `UNLEASH_CONSTRAINT_VALUES_LIMIT`              |
| [Signal](signals.md) endpoints                         |       N/A |       N/A |                5 | `UNLEASH_SIGNAL_ENDPOINTS_LIMIT`               |
| [Signal](signals.md) tokens per endpoint               |       N/A |       N/A |                5 | `UNLEASH_SIGNAL_TOKENS_PER_ENDPOINT_LIMIT`     |
| [Action](actions.md) set actions                       |       N/A |       N/A |                5 | `UNLEASH_ACTION_SET_ACTIONS_LIMIT`             |
| [Action](actions.md) sets per project                  |       N/A |       N/A |                5 | `UNLEASH_ACTION_SETS_PER_PROJECT_LIMIT`        |
| [Action](actions.md) set filters                       |       N/A |       N/A |                5 | `UNLEASH_ACTION_SET_FILTERS_LIMIT`             |
| [Action](actions.md) set filter values                 |       N/A |       N/A |                5 | `UNLEASH_ACTION_SET_FILTER_VALUES_LIMIT`       |
